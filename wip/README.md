# Cadenza WIP Notes

This directory holds non-contract working notes for future-phase planning.

Hierarchy:

- `future-support/` — follow-up opportunities implied by frozen specs or
  design documents. These notes are not commitments; they are reminders for
  roadmap review and later spec work.
- `architect/` — non-contract Architect follow-up notes that should be reviewed
  before future spec, ADR, or governance changes.

Normative contracts still live under `spec/`. ADRs still live under `docs/adr/`.
